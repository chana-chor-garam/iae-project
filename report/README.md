# IAE Project Report (LaTeX)

This folder contains the LaTeX source for the project report.

## Build

From the `iae-project/report/` directory:

```bash
pdflatex report.tex
```

This will generate `report.pdf`.

## Notes

- The report describes the diameter-estimation algorithms implemented in this repository (k-BFS, Takes–Kosters, and iFUB).
- Code references:
  - Python algorithms: `../algos/`
  - C++ algorithms: `../ifub.cpp`, `../takeskosters.cpp`
